Admin – Quản Lý Mẫu 🗂️

Thêm / sửa / xoá mẫu hiển thị ở trang menu. Xong thì bấm "Xuất templates.json" rồi upload đè file đó lên host — không cần sửa code.

Danh sách mẫu hiện tại

Thêm mẫu mới

Đây là phần gắn vào link: admin.html?template=ID — mỗi mẫu phải có ID khác nhau.
Để trống nếu chưa có ảnh, menu sẽ hiện icon mặc định.
Nhập nhiều link (mỗi dòng 1 link) để hệ thống tự random 1 link mỗi lượt tải — giúp chia đều view qua nhiều tài khoản link4m. Chỉ 1 link cũng được.
Chọn đúng hiệu ứng gốc mà admin.html sẽ dùng để tạo file cho mẫu này.
