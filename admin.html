Trang Admin – Tạo Quà Tặng 💌

Đang tải mẫu...

Chọn ảnh, sửa lời nhắn, chọn nhạc — rồi bấm "Tải file quà tặng" để có 1 file website hoàn chỉnh, sẵn sàng đem đi đăng link.

1. Ảnh

Chọn nhiều ảnh cùng lúc. Ảnh sẽ tự nén nhẹ để file không quá nặng.

2. Lời nhắn

Mỗi dòng là 1 câu sẽ hiện ngẫu nhiên trên màn hình. Xoá / thêm dòng tuỳ ý.

3. Nhạc nền

Chọn 1 file nhạc (mp3). Nhạc nhẹ, file dưới ~3MB sẽ mượt nhất.

🖼 Ảnh hoặc Video nền

Nhấn "Chọn tệp" để thêm nền. Chỉ được chọn 1 ảnh hoặc 1 video. Khuyến nghị video dưới 15MB. Có thể tạo bằng AI hoặc tải trên Internet.

👤 Ảnh đại diện

Chọn ảnh vuông để hiển thị đẹp nhất.

🐰 Nhân vật động

Chọn Sprite Sheet PNG nền trong suốt. Hệ thống sẽ tự nhận số frame. Không cần sửa code, không cần nhập số frame. Có thể tạo bằng AI hoặc tải trên Internet. Nếu không chọn sẽ dùng nhân vật mặc định.

5. Mật khẩu mở mục Quà

Chỉ dùng cho mẫu Hộp Quà Bí Mật. Người nhận phải nhập đúng mật khẩu này (không phân biệt hoa/thường, tự bỏ khoảng trắng thừa) để mở mục Quà cuối cùng — vd: ngày kỷ niệm, biệt danh...

Quy trình tải file

  1. Bấm "Tải file quà tặng" — web sẽ lưu tạm file trong trình duyệt rồi đưa bạn qua link vượt (link4m).
  2. Vượt link xong sẽ tự về trang tải file, bấm "Tải xuống ngay" để lấy file .html hoàn chỉnh.
  3. Lưu ý: phải tải trong cùng 1 trình duyệt/thiết bị vừa tạo file (vì file được lưu tạm ngay trên máy đó), và trong vòng 20 phút kể từ lúc bấm tạo.

Rất xin lỗi đã làm phiền bạn 🙏
